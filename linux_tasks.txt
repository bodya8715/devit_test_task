1) Утилита ps используется для просмотра списка процессов в Linux. Опция -A выбрать все процессы. Затем использую утилиту grep для того, чтобы отобрать нужные мне процессы по определенному критерию.
	
	   ps -A | grep "sh" | grep -Eo '^.{8}'
	   
2) Команда find – инструмент, позволяющий искать файлы. В данном примере я ищу все файлы по раширению .txt за исключением файла all.txt, что бы он не учавствовал в поиске. Запускаю команду cat для результатов поиска, она читает файлы. Потом перенаправляю вывод команды в файл all.txt.
	
	find . -name '*.txt' -not -name "all.txt" -exec cat {} >> all.txt \;
	
3) Ищу все .jpg файлы в нужной папке и для результатов поиска запускаю команду basename, которая отделяет имя файла от расширения и пути к файлу.
	
	find ./var/www/html/images/ -name "*.jpg" -exec basename -s .jpg {} \;
	
4) Создаю файл скрипта, делаю файл исполняемым, потом пишу код скрипта. Командой find нахожу нужные файлы, и для каждого результат поиска запускаю цикл. В цикле выполняю следующие действия:
	- вычисляю контрольную сумму файла
	- отделяю путь к файлу от имени файла
	- переименовую файлы, используя базовый путь к файлу и контрольную сумму в качестве имени файла.
	- записываю в dict старое и новое имя файла.
	
	   touch bash_script.sh
	   chmod +x bash_script.sh
	   
	   #!/bin/bash
	   touch dict
	   files=`find ./var/www/html/logs/ -name "*.txt"`
	   for var in $files
	   do
	   md5=`md5sum $var | grep -Eo '^[a-z0-9]{1,}'`
	   link=`echo $var | grep -Eo '.{1,}\/{1,}'`
	   mv $var $link$md5'.txt'
	   echo $var $link$md5'.txt' >> dict
	   done

	Меняю права доступа для всех остальных
	   
	   sudo chmod -R o-rwx ./var/www/html/images
	   
6) Чтобы перезаписать файл его же текущим содержимым, и при этом не создавать новый файл можно использовать "sort" с опцией -о. Данная опция позволит не создавать новый файл, а все записать в уже существующий.
	
	sort ./var/www/html/logs/access.txt -o ./var/www/html/logs/access.txt
	
7) В предыдущем задании (№5) я изменил права доступа к папке и файлам, и теперь для получения доступа к папке и поиску файлов, необходимы права суперпользователя (sudo).
	
	sudo find ./var/www/html/images -name '*cat*' -and -size +3M
