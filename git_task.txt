1) Перейти на ветку dev - git checkout dev
2) Извлечь из указанного удаленного репозитория копию текущей ветки и немедленно слить ее с локальной копией - git pull origin
3) Перейти на ветку task_4  - git checkout task_4
4) Выполнить команду - git rebase dev. Rebase позволяет перенести изменения из одной ветки в другую. Rebase сжимает все изменения в один «патч». Затем он интегрирует патч в целевую ветку.
